Dataset Overview

This dataset contains annotations of ictal chirps in intracranial EEG (iEEG) recordings from 13 patients with epilepsy. The data is derived from the Epilepsy-iEEG-Multicenter-Dataset (hosted by OpenNeuro).

üß† Research Context

This dataset contains chirp characteristics identified during ictal events in intracranial EEG (iEEG) recordings. The data includes both visual spectrograms (.png) and numerical spectrograms matrices (.mat) for time-frequency analysis of pileptic seizures. Each file represents a specific time window from electrode recordings.
